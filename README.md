# Bomberman Arcade Game
This 2-person project is an attempt to re-create the classic arcade game Bomberman.


## How to play

### Player 1
- [Arrow Keys] for movement
- [Spacebar] for bomb placement

### Player 2
- [WASD] for movement
- [/] for bomb placement


## Prerequisites
Python 3.8.3
